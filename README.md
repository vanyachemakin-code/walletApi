## Приложение WalletApi умеет:
* Увеличивать или уменьшать балланс кошелька (в зависимости от OperationType Enum).
* Обрабатывать ошибки: 
  * валидации;
  * недостатка средств; 
  * отсутствии кошелька;
  * отсутсвии конкурентного доступа;
  * таймаута блокировки; 
* Запускаться в докер контейнере (БД так же работает в контейнере).
* Настраивать различные параметры как на стороне
  приложения так и БД без пересборки контейнеров.

### P.S. Написаны миграции для базы данных с помощью liquibase и все эндпоинты покрыты тестами
